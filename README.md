# Django AI Backend

A simple Django REST API that connects to OpenAI to generate AI responses.
This project is built as a foundation for future AI work, including training
custom models and querying data-specific knowledge.

---

## Features
- Django REST Framework API
- AI-powered responses using OpenAI
- Clean and simple backend structure
- Easy to extend for custom AI models

---

## Tech Stack
- Python
- Django
- Django REST Framework
- OpenAI API
